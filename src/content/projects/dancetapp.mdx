## Overview

DanceTapp! is a mobile application built to connect dancers and dance event organizers. It empowers users to discover dance events, manage personal profiles, and engage with the dance community seamlessly. The app includes geolocation, social interactions, and event management features to cater to both dancers and organizers.

## ‚ú® Solution

### How do dancers connect today?

Our research revealed:

- 60% rely on social media for class updates but find it scattered.
- 30% use local community boards or word-of-mouth.
- 10% have no consistent way to discover dance opportunities.

### What features do dancers value most?

Based on surveys:

1. Event Discovery: Search nearby events with filters like location and type.
2. Social Features: Like, comment, and share events with ease.
3. Profile Management: Manage favorites, photos, and preferences.

### Exploring new dance platforms

DanceTapp integrates these standout features:

- Real-time Geolocation: Locate nearby events using interactive maps.
- Custom Event Management: Organizers can publish and edit events effortlessly.
- Engaging Social Tools: Users can interact with events through likes, comments, and shares.

---

### User Journey

1. Sign Up and Setup: Users create a profile and set their dance interests.
2. Discover Events: Search and filter events by location, type, and date.
3. Engage: Like, comment, and share events to build community interaction.
4. Event Management: Organizers can manage and promote their events directly through the app.

---

## üß™ User Testing

We tested the app with 20 dancers and organizers, revealing key insights:

1. Event Discovery: 18/20 users loved the geolocation and filters.
2. Social Features: 17/20 appreciated the interactive tools but suggested adding direct messaging.
3. Profile Management: Feedback emphasized the importance of customization options.
